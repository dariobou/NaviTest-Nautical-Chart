NaviTest is a full-stack desktop application developed in Java and JavaFX designed to simulate an interactive maritime navigation environment. It allows users to interact with nautical charts, use measuring tools, and complete maritime-related test questions.

## Key Features

* **Interactive Chart & Tools:** Users can interact directly with the nautical chart using built-in digital tools such as a protractor (angles), a ruler (distances), and drawing utilities (pencil, text, eraser).
* **Zoom & Navigation:** Smooth zooming and panning capabilities to explore high-resolution charts without losing detail.
* **Secure Authentication System:** Full login and registration system integrated with a relational SQL database to securely manage user accounts and session states.
* **Assessment Module:** Integrated multiple-choice test system directly linked to a database of maritime questions to evaluate the user's navigational knowledge.
* **Clean UI/UX:** Designed with a focus on usability, providing an intuitive interface for complex nautical tasks.

## Technologies Used

* **Frontend:** JavaFX (GUI design, event handling, canvas drawing)
* **Backend:** Java
* **Database:** Relational Database (SQL) for user data and test questions
* **IDE:** Apache NetBeans

## About the Project

This application was developed as part of the *Human-Computer Interaction (Interacción Persona-Computador)* coursework during my Software Engineering degree at Universitat Politècnica de València (UPV). The main goal was to design a highly interactive, user-centered interface handling multiple tools and states concurrently.
